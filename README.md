# 说明
## 技术核心
### 1. react 17.0.1 react-router-dom 5.2.0  antd 4.8.4

### 2. 使用 react hook技术

### 3. 项目使用react-create-app脚手架搭建, 后台使用java开发.
### 4. 密码加密采用JSEncrypt加密

### 5. 采用axios请求,并进行简单的封装.

## 功能
### 目前已有功能有: 登录, 登出, 修改密码,用户管理, 日志操作, 导出excel等.
### 首页继续完善中 其他功能待开发中

# 项目打开方式
## 安装依赖
    npm install
    或者
    yarn install 
## 启动项目
    npm run start 
    或者
    yarn start

## 打包方式
    npm run build
    或者
    yarn build
## 使用docker
### docker打包项目
    增加docker打包配置,使用如下:
    sh run.sh
### 构建docker基础镜像
    buildDockerBase文件夹
    sh build.sh
